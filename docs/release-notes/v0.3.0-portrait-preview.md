# v0.3.0 — Portrait Preview

This is the first save point for the portrait-first Android build.

## What changed

- Reworked the launcher around two clear choices: offline play and Steam sign-in.
- Replaced the launcher’s unused lower space with a lightweight, code-drawn background.
- Kept secondary actions behind compact disclosures instead of showing every control at once.
- Improved keyboard and safe-area behavior around Steam sign-in.
- Removed the Early Access disclaimer from mobile startup.
- Reflowed the in-game HUD for portrait screens, including the top bar, combat hand, energy, piles, and End Turn button.
- Removed the blue-grey combat gutter and the grey tray behind the hand.
- Kept player and enemy placement on the vanilla combat plane.
- Constrained Continue details and common tutorial panels to the portrait viewport.
- Prevented the in-run settings screen from colliding with the combat top bar.

## Device validation

Validated on a 1440 × 3168 Android device through the following path:

1. Launcher mode selection
2. Offline launch
3. Main menu startup without the Early Access disclaimer
4. Continue long-press details
5. Continue into combat
6. Combat HUD, hand, piles, energy, and End Turn placement
7. Pause menu and in-run settings

The release APK is version code `30045` and version name `0.3.0-portrait-preview`.
